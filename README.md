# 🍚 Lunch Mate

팀원들과 식사 일정을 미리 공유하는 달력 스케줄러. 약속이 있어 같이 못 먹는 날을 서로 미리 알 수 있게 합니다.

## 기능

- **월간 달력** — 상단에 월 표시, ◀ 이전달 / 다음달 ▶ 이동
- **날짜 칸당 최대 9명** — 3×3 그리드로 이름 버튼 배치 (각자 고유 배경색 + 검정 텍스트)
- **왼쪽 사이드바**
  - 팀원 추가 / 삭제 (최대 9명)
  - **전체 채우기** / **전체 빼기** (현재 보는 달 기준)
  - **반복 일정** — 매주 특정 요일에 팀원 자동 배치 (종료일까지)
- **변경 이력** — 하단에 모든 변경 기록
- **팀 공유** — Vercel KV에 저장되어 팀원 누구나 같은 화면을 봅니다

## 로컬 실행

```bash
npm install
npm run dev
```

KV 환경변수가 없으면 임시 파일(`/tmp`)에 저장되어 로컬에서도 바로 동작합니다.

## Vercel 배포

1. 이 저장소를 [Vercel](https://vercel.com/new) 에서 **Import**
2. Storage 탭에서 **KV (Upstash Redis)** 스토어를 생성하고 프로젝트에 **Connect**
   - `KV_REST_API_URL`, `KV_REST_API_TOKEN` 환경변수가 자동 주입됩니다
3. **Deploy**

환경변수가 연결되면 데이터가 KV에 저장되어 팀원 간 실시간 공유됩니다.
